# 贡献说明（Contributing）

本项目为**个人作品 / 面试展示项目**，目前**不接受外部代码贡献**。

不过非常欢迎交流与反馈：
- 发现 Bug 或有改进想法，欢迎提交 [Issue](../../issues)。
- 想了解架构设计与「人机协作（Human-in-the-Loop）」开发流程，可阅读 [README](README.md)。

## 本地开发约定
- 提交前请运行测试：`pytest`（应全部通过）。
- 提交信息建议清晰描述「做了什么、为什么」。
- 不要提交数据库文件、`node_modules`、打包产物（已在 `.gitignore` 中排除）。

感谢你的关注 🙏
